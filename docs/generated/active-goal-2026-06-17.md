# Active Goal

Date: 2026-06-17

## Goal

Build a truth-first, low-token, fully repeatable information-acquisition system that can close every real repo information gap across all 50 states by relying on deterministic source packs, lightweight scrapers, strict validation, conservative locality semantics, saved artifacts, and compact audits instead of chat-heavy manual review.

## Operating Constraints

- minimize token use by turning repeatable work into scripts, ledgers, fixtures, and reports
- go deep on information quality, not shallow on surface coverage
- never fake completeness, locality, or trust
- prefer first-party and official sources
- save full outputs to files and only surface compact summaries in chat
- use Playwright only when lightweight HTTP parsing cannot do the job
- do not promote data unless it clears public-safe and locality-safe rules

## Definition Of Success

- every major information family has either a scrape-ready source pack or an authored target plan
- the pipeline can scrape, validate, stage, and audit outside Codex
- providers, nonprofits, advocates, routing, forms, and competitive-help layers are all covered by deterministic workflows
- remaining gaps are explicit, measured, and attributable to missing source reality rather than missing process
- public output stays truthful while information depth keeps increasing

## Priority Order

1. Finish missing source packs.
2. Exhaust lightweight scrape-ready targets.
3. Harden parsers and validators.
4. Fix locality semantics for providers, nonprofits, and advocates.
5. Expand metadata richness from first-party evidence.
6. Promote only safe records.
7. Re-audit and repeat.
