## 2. VA Review Instructions

For each item in the queue:
1. **Search:** Run the recommended search query to find the official state page.
2. **Retrieve PDF:** Copy the direct link to the latest PDF version of the form (e.g. ending in `.pdf`).
3. **Verify Mailing Address:** Extract the postal mailing address, fax number, or online portal where parents should submit the completed form.
4. **Draft Cover Letter:** Write a 1-page parent cover letter template to attach to the application.
5. **Draft Follow-up Script:** Write a short telephone follow-up script for parents to call the agency 10 days after mailing (e.g., "Hello, my name is [Parent Name], I am calling to check if you received the special education assessment request for my child [Child Name]...").
6. **Save:** Write the completed fields into the staged database record.
